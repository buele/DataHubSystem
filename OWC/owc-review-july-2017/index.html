Principles

User friendly

The application is oriented to Earth Observation audience. Easy to use for scientific professionals

Customizable & extendible

The application is composed of reusable, customizable and portable atomic components.

Cross-device

The application runs in every device.

Cross-browser

Most popular browsers are supported.

The cross-browser compatibility is not supported in the current version of OWC.

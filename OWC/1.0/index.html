OWC 1.0



November 2017
Open Web Components is an Open Source front-end client
for Earth Observation data dissemination field
customizable, extendable, modular
to develop and to integrate a new components each other

In other words OWC is a collection of modular, customizable and extensible components, to create web applications for EO

Based on web-components standard, supported at the moment only on Opera and Chrome

Web Components is a W3C standard proposed by Google, to create new atomic HTML tag (objects), with a closed scope for CSS and JS

The Version 1.0 is just relased with the basic framework components:

  • ▸ Main menu
  • ▸ Authentication
  • ▸ Authorization
  • ▸ Networking
  • ▸ Window Manager
  • ▸ Message Broker
and specific EO components:

  • ▸ Advanced Search product
  • ▸ Map
  • ▸ Product list
  • ▸ Product detail
  • ▸ Ingestion distribution calendar
  • ▸ DHuS Admin panels

Every component is customizable and translatable in different languages.

OWC Project Facts

OWC was born on 2015 as Research&Development project to explore Web Components Technologies to implement a Single Page Application front-end of Data Hub System

After the R/D phase the project has became an official Data Hub System module in the 4th May of 2016 at 10:43:45, managed following the official Data Hub System Development workflow
